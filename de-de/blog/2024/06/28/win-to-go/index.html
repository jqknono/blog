Windows To Go Übersicht

Windows To Go ist eine Funktion der Windows Enterprise- und Education-Edition. Die meisten privaten Nutzer verwenden die Home-Edition, die diese Funktion nicht besitzt. Sie ermöglicht es, ein portables Windows-System von einem USB-Stick oder einer Festplatte zu starten. Windows To Go wurde nicht entwickelt, um herkömmliche Arbeitswerkzeuge zu ersetzen. Ihr Hauptziel ist es, Personen mit häufig wechselnden Arbeitsräumen effizienter zu machen. Bevor man Windows To Go verwendet, muss man folgende Punkte beachten:

  • Unterschiede zwischen Windows To Go und herkömmlicher Windows-Installation;
  • Arbeiten mit Windows To Go unterwegs;
  • Vorbereitung der Installation von Windows To Go;
  • Hardware-Anforderungen.

Unterschiede zwischen Windows To Go und herkömmlicher Windows-Installation

Die Arbeitsumgebung von Windows To Go ähnelt fast der von herkömmlichem Windows, mit folgenden Unterschieden:

  • Alle anderen Festplatten des Rechners sind standardmäßig offline, außer dem verwendeten USB-Stick. Das bedeutet, sie sind im Datei-Explorer nicht sichtbar, um die Datensicherheit zu schützen. [J] Man kann jedoch immer noch andere Festplatten sichtbar machen und die darin enthaltenen Dateien bearbeiten.
  • TPM (Trusted Platform Module) ist nicht verfügbar. Das TPM-Modul bindet sich an einen bestimmten Rechner, um Unternehmensdaten zu schützen. [J] Die meisten privaten Computer besitzen kein TPM-Modul, aber wenn Ihr Geschäftscomputer bereits in die Unternehmensdomäne eingebunden ist, sollten Sie Windows To Go auf diesem Computer nicht verwenden, sonst sollten Sie vorher Ihren Lebenslauf für den nächsten Job vorbereiten.
  • Der Ruhezustand von Windows To Go ist standardmäßig deaktiviert, kann jedoch über Gruppenrichtlinien aktiviert werden. [J] Viele Rechner trennen die Verbindung zu USB-Geräten im Ruhezustand, wodurch die Wiederherstellung aus dem Ruhezustand nicht möglich ist. Das ist verständlich, Microsoft hat dies bereits berücksichtigt, daher ist es nicht nötig, diese Einstellung zu ändern.
  • Die Wiederherstellungsfunktion von Windows ist deaktiviert. Wenn das System Probleme hat, muss Windows neu installiert werden.
  • Zurücksetzen auf Werkseinstellungen ist nicht verfügbar, Zurücksetzen von Windows ist nicht verfügbar.
  • Aktualisierungen sind nicht verfügbar. Windows bleibt auf der installierten Version, kann nicht von Windows 7 auf 8 aktualisiert werden oder von Windows 10 Red Stone 1 auf Red Stone 2.

Arbeiten mit Windows To Go unterwegs

Windows To Go kann zwischen mehreren Rechnern wechseln, das System entscheidet automatisch über die benötigten Treiber. Einige Anwendungen, die stark mit der Systemhardware verbunden sind, können möglicherweise nicht ausgeführt werden. [J] Zum Beispiel das Thinkpad-Touchpad-Einstellungsprogramm, das Fingerabdruckerkennungs-Setup-Programm usw.

Vorbereitung der Installation von Windows To Go

Man kann System Center Configuration Manager oder Standardbereitstellungswerkzeuge von Windows wie DiskPart oder Deployment Image Servicing and Management (DISM) verwenden. Folgende Fragen sollten beachtet werden:

  • Gibt es Treiber, die in das Windows To Go-Image eingefügt werden müssen?
  • Wie speichert und synchronisiert man Daten beim Wechsel zwischen verschiedenen Rechnern?
  • 32-Bit oder 64-Bit? [J] Neue Rechner unterstützen alle 64-Bit, 64-Bit-Prozessoren können auch 32-Bit-Systeme ausführen, 32-Bit-Prozessoren können keine 64-Bit-Systeme ausführen. 64-Bit-Systeme beanspruchen mehr Festplatten- und Arbeitsspeicherplatz. Wenn die zu migrierenden Rechner nur 32-Bit-Prozessoren besitzen oder weniger als 4 GB RAM haben, empfiehlt es sich, ein 32-Bit-System zu verwenden.
  • Welche Auflösung sollte eingestellt werden, wenn man von einem Netzwerk außerhalb des Unternehmensnetzwerks aus remote verbunden ist?

Hardware-Anforderungen

USB-Festplatte oder USB-Stick

Windows To Go wurde speziell für folgende Geräte optimiert, um die Anforderungen zu erfüllen, darunter:

  • Optimierung des hohen zufälligen Lese-/Schreibzugriffs auf USB-Geräte, um den täglichen Betrieb flüssiger zu gestalten.
  • Start von Windows 7 und neueren Systemen auf zertifizierten Geräten.
  • Auch beim Ausführen von Windows To Go genießen USB-Geräte den Herstellergarantiesupport. [J] Es wird nicht gesagt, dass der Computer, in den der USB-Stick gesteckt wird, Garantieansprüche hat.

Nicht zertifizierte USB-Geräte unterstützen Windows To Go nicht. [J] Ob es funktioniert, kann man ausprobieren, wenn es nicht klappt, weiß man auch warum. [J] Außerdem gibt es im Internet alternative Methoden, den Hersteller und das Modell des USB-Sticks zu ändern, um die Unterstützung zu erzwingen, dies wird hier nicht weiter ausgeführt.

Host-Computer

  • Zertifizierung für Windows 7 und neuere Systeme.
  • Computer mit Windows RT-System werden nicht unterstützt.
  • Apple Mac-Computer werden nicht unterstützt. [J] Obwohl es im Internet viele Berichte über die Erfahrungen gibt, Windows To Go auf Mac zu betreiben, sagt die offizielle Dokumentation klar, dass Mac-Szenarien nicht unterstützt werden.
