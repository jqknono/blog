  1. еҚҡе®ў

Monday, January 01, 0001

  1. еҲҶжһҗз”ЁжҲ·иҜ·жұӮпјҡ

    • иҫ“е…Ҙпјҡ дёҖзҜҮдёӯж–ҮеҚҡе®ўж–Үз« пјҲз®ҖдҪ“пјүпјҢжҜ”иҫғдәҶеҗ„з§Қ DNS еҠ еҜҶеҚҸи®®пјҲPlain DNSгҖҒDoTгҖҒDoHгҖҒDoQпјүгҖӮеҶ…е®№йҮҮз”Ё Markdown ж јејҸпјҢеҢ…еҗ« front matterгҖҒж ҮйўҳгҖҒеҲ—иЎЁгҖҒMermaid еӣҫиЎЁгҖҒиЎЁж је’Ңй“ҫжҺҘгҖӮ
    • д»»еҠЎпјҡ е°ҶеҶ…е®№д»Һз®ҖдҪ“дёӯж–ҮпјҲzh-cnпјүзҝ»иҜ‘дёәз№ҒдҪ“дёӯж–ҮпјҲzh-twпјүгҖӮ
    • йҷҗеҲ¶жқЎд»¶пјҡ
      • дёҘж јйҒөе®Ҳж јејҸпјҲJSON/XML/Markdown з»“жһ„пјүгҖӮ
      • еӨ„зҗҶд»Јз Ғеқ—пјҡдёҚиҰҒзҝ»иҜ‘ж ҮиҜҶз¬ҰпјҲmermaid, bash зӯүпјүгҖӮ
      • дёҚиҰҒи§ЈйҮҠпјҢдёҚиҰҒеҠ еүҚзјҖ/еҗҺзјҖгҖӮ
      • е…·дҪ“зҡ„ Front Matter и§„еҲҷпјҡзҝ»иҜ‘ tagsгҖҒcategoriesгҖҒdescriptionпјҢдёҚиҰҒзҝ»иҜ‘ slugпјҲиҷҪз„¶иҝҷйҮҢжІЎжңү slug еӯ—ж®өпјҢдҪҶеҖјеҫ—жіЁж„ҸпјүгҖӮtitle йҖҡеёёйңҖиҰҒзҝ»иҜ‘д»ҘеҢ№й…ҚжӯЈж–ҮиҜӯиЁҖгҖӮ

  2. жЈҖжҹҘзҝ»иҜ‘еҝ…иҰҒжҖ§пјҲйҖ»иҫ‘жЈҖжҹҘпјүпјҡ

    • еҶ…е®№жҳҜ Markdown ж–ҮжЎЈгҖӮ
    • е®ғеҢ…еҗ«еӨ§йҮҸйңҖиҰҒжң¬ең°еҢ–зҡ„иҮӘз„¶иҜӯиЁҖж–Үжң¬гҖӮ
    • Front matter еҢ…еҗ«йңҖиҰҒзҝ»иҜ‘зҡ„е…ій”®еӯ—пјҲtags, categories, descriptionпјүгҖӮ
    • з»“и®әпјҡ зҝ»иҜ‘жҳҜеҝ…йңҖзҡ„гҖӮ

  3. жү§иЎҢ - йҖҗж®өзҝ»иҜ‘дёҺж јејҸеҢ–пјҡ

    • Front Matterпјҡ

      • layout: blog -> дҝқз•ҷ
      • categories: ["network"] -> network жҳҜиӢұиҜӯпјҢдҝқз•ҷгҖӮ
      • tags: ["DNS", "DoH", "DoT", "QUIC"] -> зј©еҶҷ/иӢұиҜӯжңҜиҜӯпјҢдҝқз•ҷгҖӮ
      • draft: false -> дҝқз•ҷ
      • title: DNS еҠ еҜҶеҚҸи®®еҜ№жҜ”пјҡDoTгҖҒDoHгҖҒDoQ -> з®ҖдҪ“ иҪ¬ з№ҒдҪ“гҖӮ
        • еҠ еҜҶ -> еҠ еҜҶ
        • еҚҸи®® -> еҚ”иӯ°
        • еҜ№жҜ” -> е°ҚжҜ”
        • з»“жһңпјҡDNS еҠ еҜҶеҚ”иӯ°е°ҚжҜ”пјҡDoTгҖҒDoHгҖҒDoQ
      • date: ... -> дҝқз•ҷ
      • comments: true -> дҝқз•ҷ
      • giscus_comments: true -> дҝқз•ҷ
      • description: жўізҗҶ Plain DNSгҖҒDoTгҖҒDoHгҖҒDoQ зҡ„еҲҶеұӮе…ізі»гҖҒз«ҜеҸЈгҖҒжҖ§иғҪе·®ејӮдёҺйҖӮз”ЁеңәжҷҜпјҢз»ҷеҮәе®һйҷ…йҖүжӢ©дёҺй…ҚзҪ®е»әи®®гҖӮ -> з®ҖдҪ“ иҪ¬ з№ҒдҪ“гҖӮ
        • жўізҗҶ -> жўізҗҶ
        • еҲҶеұӮе…ізі» -> еҲҶеұӨй—ңдҝӮ
        • з«ҜеҸЈ -> з«ҜеҸЈпјҲжңҜиҜӯпјҢжңүж—¶жҳҜеҹ пјҢдҪҶеңЁжҠҖжңҜиҜӯеўғдёӢз«ҜеҸЈеҫҲеёёи§ҒпјҢйҖҡеёёдҝқжҢҒеҺҹж ·жҲ–жҳ е°„гҖӮеңЁ TW жҠҖжңҜж–ҮжЎЈдёӯпјҢ‘з«ҜеҸЈ’ иў«е№ҝжіӣжҺҘеҸ—пјҢ‘еҹ ’ жӣҙдј з»ҹгҖӮжҲ‘е°ҶеқҡжҢҒдҪҝз”Ёж ҮеҮҶжҠҖжңҜжңҜиҜӯпјҢйҖҡеёёжҳҜ ‘з«ҜеҸЈ’ жҲ–дёҘж јеҜ№еә”иҜӯеўғгҖӮ‘з«ҜеҸЈ’ йҖҡеёёжҳҜе®үе…Ёзҡ„пјҢдҪҶи®©жҲ‘д»¬жЈҖжҹҘдёҖдёӢзү№е®ҡиҜӯеўғгҖӮ‘еҲҶеұӨй—ңдҝӮ’ -> еұӨзҙҡй—ңдҝӮ жҲ– еҲҶеұӨй—ңдҝӮгҖӮ‘жҖ§иғҪ’ -> ‘ж•ҲиғҪ’гҖӮ‘е·®ејӮ’ -> ‘е·®з•°’гҖӮ‘йҖӮз”Ё’ -> ‘йҒ©з”Ё’гҖӮ‘е®һйҷ…’ -> ‘еҜҰйҡӣ’гҖӮ‘й…ҚзҪ®’ -> ‘й…ҚзҪ®’гҖӮ
        • иҚүзЁҝпјҡжўізҗҶ Plain DNSгҖҒDoTгҖҒDoHгҖҒDoQ зҡ„еҲҶеұӨй—ңдҝӮгҖҒз«ҜеҸЈгҖҒж•ҲиғҪе·®з•°иҲҮйҒ©з”Ёе ҙжҷҜпјҢзөҰеҮәеҜҰйҡӣйҒёж“ҮиҲҮй…ҚзҪ®е»әиӯ°гҖӮ
      • weight: 100 -> дҝқз•ҷ

    • еҶ…е®№ - ж Үйўҳ 2пјҡеҗҚиҜҚйҖҹи§Ҳ

      • еҗҚиҜҚ -> еҗҚи©һ
      • йҖҹи§Ҳ -> йҖҹиҰҪ

    • еҲ—иЎЁпјҡ

      • Plain DNSпјҡжҳҺж–Ү DNSпјҢй»ҳи®ӨдҪҝз”Ё UDP/53пјҢеҝ…иҰҒж—¶з”Ё TCP/53пјҲеҰӮе“Қеә”иў«жҲӘж–ӯгҖҒеҢәеҹҹдј йҖҒзӯүпјүгҖӮ
        • жҳҺж–Ү -> жҳҺж–Ү
        • й»ҳи®Ө -> й җиЁӯ
        • еҝ…иҰҒж—¶ -> еҝ…иҰҒжҷӮ
        • е“Қеә” -> еӣһжҮү
        • жҲӘж–ӯ -> жҲӘж–·
        • еҢәеҹҹдј йҖҒ -> еҚҖеҹҹеӮійҖҒ
      • DoTпјҡDNS over TLSпјҢдҪҝз”Ё TLS д№ӢдёҠзҡ„ TCPпјҢй»ҳи®Өз«ҜеҸЈ 853пјҲRFC 7858/8310пјүгҖӮ -> й җиЁӯз«ҜеҸЈ
      • DoHпјҡDNS over HTTPSпјҢеҹәдәҺ HTTPSпјҲHTTP/2 жҲ– HTTP/3пјүпјҢй»ҳи®Өз«ҜеҸЈ 443пјҲRFC 8484пјүгҖӮ -> еҹәж–јгҖҒй җиЁӯз«ҜеҸЈ
      • DoQпјҡDNS over QUICпјҢеҹәдәҺ QUIC + TLS 1.3пјҢй»ҳи®Өз«ҜеҸЈ UDP/853пјҲRFC 9250пјҢIANA е·ІеҲҶй…ҚеңЁ 853/udpпјүгҖӮ -> еҹәж–јгҖҒй җиЁӯз«ҜеҸЈ

    • еҶ…е®№ - ж Үйўҳ 2пјҡеҲҶеұӮе…ізі»пјҲз®ҖеҢ– TCP/IP жЁЎеһӢпјү

      • еҲҶеұӮ -> еҲҶеұӨ
      • е…ізі» -> й—ңдҝӮ
      • з®ҖеҢ– -> з°ЎеҢ–
      • жЁЎеһӢ -> жЁЎеһӢ

    • еҲ—иЎЁпјҡ

      • еә”з”ЁеұӮ -> жҮүз”ЁеұӨ
      • е°ҒиЈ… -> е°ҒиЈқ
      • е®үе…ЁеұӮ -> е®үе…ЁеұӨ
      • дј иҫ“еұӮ -> еӮіијёеұӨ
      • зҪ‘з»ңеұӮ -> з¶Іи·ҜеұӨ
      • й“ҫи·ҜеұӮ -> йҸҲи·ҜеұӨпјҲжҲ– йҸҲзөҗеұӨпјҢдҪҶеңЁз®ҖдҪ“дёӯжҳҜ ‘й“ҫи·ҜеұӮ’ дё”йҖҡеёёдҝқжҢҒж ҮеҮҶпјҢж ҮеҮҶз№ҒдҪ“йҖҡеёёжҳҜ йҸҲи·ҜеұӨ жҲ– ж•ёж“ҡйҸҲи·ҜеұӨгҖӮжҲ‘е°ҶдҪҝз”Ё йҸҲи·ҜеұӨпјүгҖӮ
      • зү©зҗҶеұӮ -> зү©зҗҶеұӨ
      • д»ҘеӨӘзҪ‘ -> д»ҘеӨӘз¶Іи·Ҝ
      • еҸҢз»һзәҝ/е…үзәӨ/ж— зәҝ -> йӣҷзөһз·ҡ/е…үзә–/з„Ўз·ҡ

    • еҶ…е®№ - ж Үйўҳ 2пјҡиҰҒзӮ№

      • иҰҒзӮ№ -> иҰҒй»һ

    • еҲ—иЎЁпјҡ

      • жҖ§иғҪ -> ж•ҲиғҪ

    • еҶ…е®№ - ж Үйўҳ 2пјҡеҹәзЎҖзҹҘиҜҶдёҺеӢҳиҜҜ
