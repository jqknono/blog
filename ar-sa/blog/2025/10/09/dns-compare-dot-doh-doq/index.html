  • Plain DNS: DNS ЩҶШөЩҠ ЩҲШ§Ш¶ШӯШҢ ЩҠШіШӘШ®ШҜЩ… UDP/53 Ш§ЩҒШӘШұШ§Ш¶ЩҠЩӢШ§ШҢ Щ…Ш№ Ш§Щ„Ш§ЩҶШӘЩӮШ§Щ„ ШҘЩ„Щү TCP/53 Ш№ЩҶШҜ Ш§Щ„ШӯШ§Ш¬Ш© (ЩғШ§Щ„Ш§ШіШӘШ¬Ш§ШЁШ§ШӘ Ш§Щ„Щ…ЩӮШ·ЩҲШ№Ш©ШҢ ЩҶЩӮЩ„ Ш§Щ„Щ…ЩҶШ·ЩӮШ©ШҢ ШҘЩ„Ш®).
  • DoT: DNS over TLSШҢ ЩҠШіШӘШ®ШҜЩ… TCP ЩҒЩҲЩӮ TLSШҢ Щ…ЩҶЩҒШ° Ш§ЩҒШӘШұШ§Ш¶ЩҠ 853 (RFC 7858/8310).
  • DoH: DNS over HTTPSШҢ ЩҠШіШӘШ®ШҜЩ… HTTPS (HTTP/2 ШЈЩҲ HTTP/3)ШҢ Щ…ЩҶЩҒШ° Ш§ЩҒШӘШұШ§Ш¶ЩҠ 443 (RFC 8484).
  • DoQ: DNS over QUICШҢ ЩҠШіШӘШ®ШҜЩ… QUIC + TLS 1.3ШҢ Щ…ЩҶЩҒШ° Ш§ЩҒШӘШұШ§Ш¶ЩҠ UDP/853 (RFC 9250ШҢ Щ…Ш®ШөШө ЩҒЩҠ IANA Ш№Щ„Щү 853/udp).

Ш§Щ„Ш№Щ„Ш§ЩӮШ§ШӘ Ш§Щ„Ш·ШЁЩӮЩҠШ© (ЩҶЩ…ЩҲШ°Ш¬ TCP/IP Щ…ШЁШіШ·)

  • Ш·ШЁЩӮШ© Ш§Щ„ШӘШ·ШЁЩҠЩӮ: HTTPШҢ HTTPSШҢ DNS (DoH Ш¶Щ…ЩҶ ШӘШәЩ„ЩҠЩҒ HTTPS ЩҒЩҠ Ш·ШЁЩӮШ© Ш§Щ„ШӘШ·ШЁЩҠЩӮ)
  • Ш·ШЁЩӮШ© Ш§Щ„ШЈЩ…Ш§ЩҶ: TLS (ШӘЩҲЩҒШұ Ш§Щ„ШӘШҙЩҒЩҠШұ Щ„ЩҖ TCP ШЈЩҲ QUIC)
  • Ш·ШЁЩӮШ© Ш§Щ„ЩҶЩӮЩ„: TCPШҢ UDPШҢ QUIC
  • Ш·ШЁЩӮШ© Ш§Щ„ШҙШЁЩғШ©: IP
  • Ш·ШЁЩӮШ© Ш§Щ„ШұШЁШ·: Ethernet
  • Ш§Щ„Ш·ШЁЩӮШ© Ш§Щ„ЩҒЩҠШІЩҠШ§ШҰЩҠШ©: ЩғШ§ШЁЩ„ Щ…Ш¬ШҜЩҲЩ„/ШЈЩ„ЩҠШ§ЩҒ/Щ„Ш§ШіЩ„ЩғЩҠ

Ш§Щ„ЩҶЩӮШ§Ш· Ш§Щ„ШЈШіШ§ШіЩҠШ©

  • Plain DNS ЩҠШ№Щ…Щ„ ЩҒЩҲЩӮ UDP/TCP ШҜЩҲЩҶ ШӘШҙЩҒЩҠШұ.
  • DoT = TCP + TLS + DNS (Щ…ЩҶЩҒШ° Щ…Ш®ШөШө 853).
  • DoH = TCP/QUIC + TLS + HTTP(S) + DNS (ЩҠШіШӘШ®ШҜЩ… 443ШҢ Щ…ШҙШӘШұЩғ Щ…Ш№ HTTPS Ш§Щ„Ш№Ш§ШҜЩҠ).
  • DoQ = QUIC + TLS 1.3 + DNS (Щ…ЩҶЩҒШ° Щ…Ш®ШөШө UDP/853).

Щ…Ш№Щ„ЩҲЩ…Ш§ШӘ ШЈШіШ§ШіЩҠШ© ЩҲШӘШөШӯЩҠШӯ ШЈШ®Ш·Ш§ШЎ ШҙШ§ШҰШ№Ш©

  • Plain DNS ЩҠШіШӘШ®ШҜЩ… UDP/53 Ш§ЩҒШӘШұШ§Ш¶ЩҠЩӢШ§ШҢ ЩҲШ№ЩҶШҜ ШӯШҜЩҲШ« ЩӮШ·Ш№ ЩҒЩҠ Ш§Щ„Ш§ШіШӘШ¬Ш§ШЁШ© (bit TC) ШЈЩҲ Ш§Щ„ШӯШ§Ш¬Ш© Щ„ЩҶЩӮЩ„ Щ…ЩҲШ«ЩҲЩӮШҢ ЩҠЩҶШӘЩӮЩ„ ШҘЩ„Щү TCP/53.
  • DoT ЩҠШЁЩҶЩҠ ЩҶЩҒЩӮ TLS ЩҒЩҲЩӮ TCP Щ„ЩҶЩӮЩ„ ШӯШІЩ… DNSШҢ Щ…ЩҶЩҒШ° Ш§ЩҒШӘШұШ§Ш¶ЩҠ 853Шӣ ЩҠЩ…ЩғЩҶ ШҘШ№Ш§ШҜШ© Ш§ШіШӘШ®ШҜШ§Щ… Ш§Щ„Ш§ШӘШөШ§Щ„ Ш§Щ„Ш·ЩҲЩҠЩ„ Щ„Ш®ЩҒШ¶ ШӘЩғЩ„ЩҒШ© Ш§Щ„ШұШЁШ·.
  • DoH ЩҠЩҸШҜШ®Щ„ DNS ЩғЩ…ЩҲШұШҜ Ш¶Щ…ЩҶ HTTPS (application/dns-message)ШҢ ШәШ§Щ„ШЁЩӢШ§ ШЁШ§ШіШӘШ®ШҜШ§Щ… HTTP/2 ШЈЩҲ HTTP/3ШҢ Щ…ЩҶЩҒШ° 443ШҢ ЩҠШіЩҮЩ„ Ш§Щ„Ш§Ш®ШӘЩ„Ш§Ш· Щ…Ш№ HTTPS Ш§Щ„Ш№Ш§ШҜЩҠ.
  • DoQ ЩҠШіШӘШ®ШҜЩ… QUIC Щ…ШЁШ§ШҙШұШ© (Щ…ШЁЩҶЩҠШ© Ш№Щ„Щү UDP) Щ„ШӯЩ…Щ„ DNSШҢ ЩҲШӘШӘЩ…ЩҠШІ ШЁШӘШЈШ®ЩҠШұ Щ…ЩҶШ®ЩҒШ¶ ЩҲШӘШ¬ЩҶШЁ Ш§ШІШҜШӯШ§Щ… Ш§Щ„Ш·Ш§ШЁЩҲШұШҢ Щ„ЩғЩҶ ШӘШәШ·ЩҠШ© Ш§Щ„ШЁЩҠШҰШ© Щ„Ш§ ШӘШІШ§Щ„ ЩҒЩҠ ЩҶЩ…ЩҲ.
  • Ш§Щ„ЩӮЩҲЩ„ ШЁШЈЩҶ “QUIC ШЈШіШұШ№ ШҜШ§ШҰЩ…ЩӢШ§ ШЁЩҶШіШЁШ© X%” Щ„ЩҠШі ШҜЩӮЩҠЩӮЩӢШ§Шӣ Ш§Щ„ШЈШҜШ§ШЎ Ш§Щ„ЩҒШ№Щ„ЩҠ ЩҠШ№ШӘЩ…ШҜ Ш№Щ„Щү ШӯШ§Щ„Ш© Ш§Щ„ШҙШЁЩғШ© (ЩҒЩӮШҜШ§ЩҶ Ш§Щ„ШӯШІЩ…ШҢ Ш§Щ„ШӘШ°ШЁШ°ШЁШҢ ШІЩ…ЩҶ Ш§Щ„Ш§ЩҶШӘЩӮШ§Щ„) ЩҲЩӮШҜШұШ© ШҘШ№Ш§ШҜШ© Ш§ШіШӘШ®ШҜШ§Щ… Ш§Щ„Ш§ШӘШөШ§Щ„ ЩҲШӘЩҒШ§ШөЩҠЩ„ Ш§Щ„ШӘЩҶЩҒЩҠШ° ЩҲЩҶШҙШұ Ш§Щ„Ш®Ш§ШҜЩ….
  • DoH Щ„ЩҠШі “ЩҲШ¶Ш№ DNS ШҜШ§Ш®Щ„ HTTP ЩҠШ¬Ш№Щ„ЩҮ ШЈШіШұШ№/ШЈШЁШ·ШЈ ШЁШ§Щ„Ш¶ШұЩҲШұШ©”ШҢ Ш§Щ„ШЈШҜШ§ШЎ ЩҠШ№ШӘЩ…ШҜ Ш№Щ„Щү ШҘШ№Ш§ШҜШ© Ш§ШіШӘШ®ШҜШ§Щ… Ш§Щ„Ш§ШӘШөШ§Щ„ШҢ Ш¬ЩҲШҜШ© Ш§Щ„ШҙШЁЩғШ©ШҢ ЩҲШ§Щ„ШӘЩҶЩҒЩҠШ°Шӣ ЩҒЩҠ ЩғШ«ЩҠШұ Щ…ЩҶ Ш§Щ„ШӯШ§Щ„Ш§ШӘ ЩҠЩғЩҲЩҶ DoH/3 ЩҲ DoT Щ…ШӘЩ…Ш§Ш«Щ„ЩҠЩҶ ШЈЩҲ ШЈЩҒШ¶Щ„.
  • DoT ЩҠЩ…ЩғЩҶЩҮ Ш§Щ„ШӘШӯЩӮЩӮ Щ…ЩҶ Ш§ШіЩ… Ш§Щ„Щ…Ш¶ЩҠЩҒ ШЁШ§ШіШӘШ®ШҜШ§Щ… SNIШӣ DoH ЩҠШ№ШӘЩ…ШҜ Ш№Щ„Щү Ш§Щ„ШӘШӯЩӮЩӮ Ш§Щ„ЩӮЩҠШ§ШіЩҠ Щ…ЩҶ ШҙЩҮШ§ШҜШ© HTTPS ЩҲЩ…Ш·Ш§ШЁЩӮШ© Ш§ШіЩ… Ш§Щ„Щ…Ш¶ЩҠЩҒ.
  • DNS Ш§Щ„Щ…ШҙЩҒШұ Щ„Ш§ ЩҠЩ…ЩҶШ№ Ш§Щ„ШӘЩҶШөШӘ ЩҲШ§Щ„ШӘЩ„Ш§Ш№ШЁ Ш№Щ„Щү Ш®Ш· Ш§Щ„Ш§ШӘШөШ§Щ„ ЩҒЩӮШ·ШҢ ЩҲЩ„Ш§ ЩҠШ№ЩҶЩҠ “Ш§Щ„Ш®ШөЩҲШөЩҠШ© Ш§Щ„ЩғШ§Щ…Щ„Ш©”. ЩҶШёШұШ© Ш№Ш§Щ…Ш© Ш№Щ„Щү Ш§Щ„Щ…ЩӮШ§ШұЩҶШ©

Ш§Щ„ШЈШҜШ§ШЎ ЩҲШ§Щ„ШӘШЈШ®ЩҠШұ

  • ШҘШ№Ш§ШҜШ© Ш§ШіШӘШ®ШҜШ§Щ… Ш§Щ„Ш§ШӘШөШ§Щ„: DoT/DoH/DoQ ЩҠЩ…ЩғЩҶЩҮШ§ Ш¬Щ…ЩҠШ№ЩӢШ§ ШҘШ№Ш§ШҜШ© Ш§ШіШӘШ®ШҜШ§Щ… Ш§Щ„Ш§ШӘШөШ§Щ„ Ш§Щ„Ш·ЩҲЩҠЩ„ Щ„Ш®ЩҒШ¶ ШӘЩғЩ„ЩҒШ© Ш§Щ„ШұШЁШ·Шӣ DoH/2 ЩҲ DoH/3 ЩҲ DoQ ШӘШҜШ№Щ… ШЈЩҠШ¶ЩӢШ§ Ш§Щ„ШӘШ№ШҜШҜЩҠШ© ШҜШ§Ш®Щ„ Ш§ШӘШөШ§Щ„ ЩҲШ§ШӯШҜ.
  • Ш§ШІШҜШӯШ§Щ… Ш§Щ„Ш·Ш§ШЁЩҲШұ: TCP ЩҠШ№Ш§ЩҶЩҠ Щ…ЩҶ “Ш§ШІШҜШӯШ§Щ… Ш§Щ„Ш·Ш§ШЁЩҲШұ” Ш№Щ„Щү Щ…ШіШӘЩҲЩү Ш§Щ„ШӘШ·ШЁЩҠЩӮШӣ HTTP/2 ЩҠШ®ЩҒЩҒ Ш°Щ„Щғ Ш№ШЁШұ Ш§Щ„ШӘШ№ШҜШҜЩҠШ© Щ„ЩғЩҶЩҮ Щ„Ш§ ЩҠШІШ§Щ„ Ш№ШұШ¶Ш© Щ„ЩҒЩӮШҜШ§ЩҶ TCPШҢ ЩҒЩҠ ШӯЩҠЩҶ ШЈЩҶ QUIC (DoH/3ШҢ DoQ) ЩҠШӘШ¬ЩҶШЁ Ш§ШІШҜШӯШ§Щ… Ш§Щ„Ш·Ш§ШЁЩҲШұ Ш№Щ„Щү Щ…ШіШӘЩҲЩү Ш§Щ„ЩҶЩӮЩ„ШҢ ЩҲЩҮЩҲ ШЈЩҒШ¶Щ„ ЩҒЩҠ ШҙШЁЩғШ§ШӘ Ш№Ш§Щ„ЩҠШ© Ш§Щ„ЩҒЩӮШҜ/Ш§Щ„Щ…ШӘЩҶЩӮЩ„Ш©.
  • ШӘШЈШ®ЩҠШұ Ш§Щ„ШӯШІЩ…Ш© Ш§Щ„ШЈЩҲЩ„Щү: Ш№ЩҶШҜ Ш§Щ„Ш§ШӘШөШ§Щ„ Ш§Щ„ШЈЩҲЩ„ЩҠШҢ DoT ЩҠШӯШӘШ§Ш¬ ШҘЩ„Щү TCP+TLSШӣ DoH/2 Щ…ШҙШ§ШЁЩҮШӣ DoH/3/DoQ Щ…ШЁЩҶЩҠ Ш№Щ„Щү QUICШҢ ШЈШіШұШ№ ЩҒЩҠ ШҘШ№Ш§ШҜШ© Ш§Щ„Ш§ШӘШөШ§Щ„ ЩҲШ§Щ„Ш§ЩҶШӘЩӮШ§Щ„. Ш§Щ„Ш§Ш®ШӘЩ„Ш§ЩҒ ЩҒЩҠ Ш§Щ„ШЈШӯЩ…Ш§Щ„ Ш§Щ„Ш·ЩҲЩҠЩ„Ш© ЩҠШ№ШӘЩ…ШҜ ШЈЩғШ«Шұ Ш№Щ„Щү Ш§Щ„ШӘЩҶЩҒЩҠШ° ЩҲШёШұЩҲЩҒ Ш§Щ„ШҙШЁЩғШ©.
  • Ш§Щ„ЩӮШҜШұШ© Ш№Щ„Щү Ш§Щ„ЩҲШөЩҲЩ„: DoH ЩҠШіШӘШ®ШҜЩ… Щ…ЩҶЩҒШ° 443ШҢ Ш§Щ„ШЈЩӮЩ„ Ш№ШұШ¶Ш© Щ„Щ„ШӯШёШұ Ш§Щ„ШЁШіЩҠШ· ШЁШ§Щ„Щ…ЩҶЩҒШ°Шӣ DoT ЩҠШіШӘШ®ШҜЩ… 853ШҢ ШәШ§Щ„ШЁЩӢШ§ Щ…Ш§ ЩҠЩҸШӯШёШұ ШЁдёҖеҲҖеҲҮШӣ DoQ ЩҠШіШӘШ®ШҜЩ… 853/UDPШҢ ЩӮШҜ ЩҠЩҸШӯШёШұ ШЈЩҲ Щ„Ш§ ЩҠЩҸШіЩ…Шӯ ШЁЩҮ ЩҒЩҠ Ш§Щ„ЩҲЩӮШӘ Ш§Щ„ШӯШ§Щ„ЩҠ.

ШҜШ№Щ… Ш§Щ„Ш№Щ…ЩҠЩ„ ЩҲШ§Щ„ЩҶШёШ§Щ…

  • Ш§Щ„Щ…ШӘШөЩҒШӯШ§ШӘ: Ш№Ш§ШҰЩ„Ш© Chromium ЩҲ Firefox ШӘШҜШ№Щ… DoH Ш§ЩҒШӘШұШ§Ш¶ЩҠЩӢШ§ (ЩҠЩ…ЩғЩҶЩҮШ§ Ш§Щ„ШӘШұЩӮЩҠШ© ШӘЩ„ЩӮШ§ШҰЩҠЩӢШ§ ШҘЩ„Щү Щ…ШӯЩ„Щ„ ЩҠШҜШ№Щ… DoH ШЈЩҲ Ш§ШіШӘШ®ШҜШ§Щ… ЩӮШ§ШҰЩ…Ш© Щ…ШІЩҲШҜЩҠЩҶ Щ…ШҜЩ…Ш¬ЩҠЩҶ).
  • Windows: Windows 11 ЩҠШҜШ№Щ… DoH ШЈШөЩ„ЩҠЩӢШ§.
  • Android: Android 9+ ЩҠЩҲЩҒШұ “DNS Ш®Ш§Шө” (DoT Ш№Щ„Щү Щ…ШіШӘЩҲЩү Ш§Щ„ЩҶШёШ§Щ…). ШӘШәШ·ЩҠШ© DoH Ш№Щ„Щү Щ…ШіШӘЩҲЩү Ш§Щ„ЩҶШёШ§Щ… ШӘШ№ШӘЩ…ШҜ Ш№Щ„Щү Ш§Щ„ШҘШөШҜШ§Шұ/Ш§Щ„ШөШ§ЩҶШ№.
  • Щ…ЩҶШөШ§ШӘ Apple: iOS 14+/macOS 11+ ШӘШҜШ№Щ… DoT ЩҲ DoH Ш№ШЁШұ Щ…Щ„ЩҒШ§ШӘ Ш§Щ„ЩҲШөЩҒ ШЈЩҲ NetworkExtension.

ЩҶШөШ§ШҰШӯ Ш§Щ„ЩҶШҙШұ ЩҲШ§Щ„Ш§Ш®ШӘЩҠШ§Шұ

  • ШҙШЁЩғШ§ШӘ Ш№Ш§ШҜЩҠШ©/Щ…ЩӮЩҠШҜШ© (Щ…Ш«Щ„ WiвҖ‘Fi Ш№Ш§Щ…ШҢ Ш§Щ„ШӯШ§Ш¬Ш© Щ„Ш§Ш®ШӘШұШ§ЩӮ Ш§Щ„ШӯШёШұ Ш§Щ„ШЁШіЩҠШ·): ШЈЩҲЩ„ЩҲЩҠШ© Щ„ЩҖ DoH (Щ…ЩҶЩҒШ° 443)ШҢ ЩҠЩ…ЩғЩҶ ШӘЩҒШ№ЩҠЩ„ HTTP/3.
  • Ш®ШұЩҲШ¬ Щ…ЩҲШӯШҜ Ш№Щ„Щү Щ…ШіШӘЩҲЩү Ш§Щ„ЩҶШёШ§Щ… (Ш§Щ„ШұШ§ЩҲШӘШұШҢ Ш§Щ„ШЁЩҲШ§ШЁШ©ШҢ DNS Ш®Ш§Шө Android): ШЈЩҲЩ„ЩҲЩҠШ© Щ„ЩҖ DoT (853)ШҢ Щ…Ш№ ШҘШ¶Ш§ЩҒШ© DoH ЩғШ®ЩҠШ§Шұ Ш§ШӯШӘЩҠШ§Ш·ЩҠ ШҘШ°Ш§ ШіЩ…Шӯ Ш§Щ„ШҙШЁЩғШ©.
  • ШҙШЁЩғШ§ШӘ Ш№Ш§Щ„ЩҠШ© Ш§Щ„ЩҒЩӮШҜ/Ш§Щ„Щ…ШӘЩҶЩӮЩ„Ш©: ШЈЩҲЩ„ЩҲЩҠШ© Щ„ЩҖ DoH/3 ШЈЩҲ DoQ Ш§Щ„Щ…ШІЩҲШҜЩҠЩҶ ШЁЩҖ QUIC (ШӯШіШЁ ШҜШ№Щ… Ш§Щ„Щ…ШӯЩ„Щ„ ЩҲШ§Щ„Ш№Щ…ЩҠЩ„).
  • ШіЩҠЩҶШ§ШұЩҠЩҲЩҮШ§ШӘ Ш§Щ„Щ…ШӨШіШіШ©/Ш§Щ„Ш§Щ…ШӘШ«Ш§Щ„: Ш§Ш®ШӘШұ ШӯШіШЁ Ш§Щ„ШіЩҠШ§ШіШ© (DoH ЩҠЩ…ЩғЩҶ ШҜЩ…Ш¬ЩҮ Щ…Ш№ ШЁЩҶЩҠШ© HTTPS Ш§Щ„ШӯШ§Щ„ЩҠШ©Шӣ DoT ЩҠШіЩҮЩ„ Ш§Щ„ЩҒШөЩ„ Ш№ЩҶ ЩҲШ§Ш¬ЩҮШ© ШӘШӯЩғЩ… DNS).

Щ…Щ„Ш®Шө

  • Ш§Щ„ШЈЩҲЩ„ЩҲЩҠШ© Щ„ЩҖ DoH (443ШҢ Ш§Ш®ШӘШұЩӮ ЩӮЩҲЩҠ)ШҢ ШҘШ°Ш§ ЩғШ§ЩҶ Щ…ШӘШ§ШӯЩӢШ§ ЩҒЩҒШ№Щ‘Щ„ HTTP/3.
  • ШҘШ°Ш§ ЩғЩҶШӘ ШЁШӯШ§Ш¬Ш© Щ„ЩҲШӯШҜШ© Ш№Щ„Щү Щ…ШіШӘЩҲЩү Ш§Щ„ЩҶШёШ§Щ…: ШЈЩҲЩ„ЩҲЩҠШ© Щ„ЩҖ DoT (853) + Ш§ШӘШөШ§Щ„ ШҜШ§ШҰЩ…ШҢ Щ…Ш№ DoH (443) ЩғШ®ЩҠШ§Шұ Ш§ШӯШӘЩҠШ§Ш·ЩҠ Ш№ЩҶШҜ Ш§Щ„Ш¶ШұЩҲШұШ©.
  • ШҘШ°Ш§ ЩғШ§ЩҶ Щ…ШӯЩ„Щ„Щғ ЩҲШ№Щ…ЩҠЩ„Щғ ЩҠШҜШ№Щ…Ш§ЩҶ Ш°Щ„Щғ: Ш¬ШұЩ‘ШЁ DoQ (ШәШ§Щ„ШЁЩӢШ§ ШӘШ¬ШұШЁШ© ШЈЩҒШ¶Щ„ ЩҒЩҠ Ш§Щ„ШҙШЁЩғШ§ШӘ Ш§Щ„Щ…ШӘЩҶЩӮЩ„Ш©).
