  1. т▒▒ТІгт╝Д <>№╝џ
    • т▒▒ТІгт╝ДсЂ»т┐ЁжаѕсЃЉсЃЕсЃАсЃ╝сѓ┐сѓњУАесЂЎсЂ«сЂФСй┐ућесЂЋсѓїсЂЙсЂЎсђѓсЂцсЂЙсѓісђЂсѓ│сЃъсЃ│сЃЅсѓњт«ЪУАїсЂЎсѓІжџЏсЂФт┐ЁсЂџТЈљСЙЏсЂЎсѓІт┐ЁУдЂсЂ«сЂѓсѓІтђцсЂДсЂЎсђѓжђџтИИсђЂсѓ│сЃъсЃ│сЃЅсЂ«тЪ║ТюгуџёсЂфТДІТќЄсЂесЃЉсЃЕсЃАсЃ╝сѓ┐сѓњУАесЂЎсЂЪсѓЂсЂФСй┐ућесЂЋсѓїсЂЙсЂЎсђѓ
    • СЙІ№╝џcommand <filename> сЂ»сђЂсЃЋсѓАсѓцсЃФтљЇсѓњт┐ЁжаѕсЃЉсЃЕсЃАсЃ╝сѓ┐сЂесЂЌсЂдТЈљСЙЏсЂЎсѓІт┐ЁУдЂсЂїсЂѓсѓІсЂЊсЂесѓњуц║сЂЌсЂдсЂёсЂЙсЂЎсђѓСЙІсЂѕсЂ░ command file.txt сЂ«сѓѕсЂєсЂФсЂЌсЂЙсЂЎсђѓ
  2. УДњТІгт╝Д []№╝џ
    • УДњТІгт╝ДсЂ»сѓфсЃЌсѓисЃДсЃ│сЂ«сЃЉсЃЕсЃАсЃ╝сѓ┐сѓњУАесЂЎсЂ«сЂФСй┐ућесЂЋсѓїсЂЙсЂЎсђѓсЂцсЂЙсѓісђЂсѓ│сЃъсЃ│сЃЅсѓњт«ЪУАїсЂЎсѓІжџЏсЂФжЂИТіъуџёсЂФТЈљСЙЏсЂДсЂЇсѓІтђцсЂДсЂЎсђѓжђџтИИсђЂсѓ│сЃъсЃ│сЃЅсЂ«сѓфсЃЌсѓисЃДсЃ│сЃЉсЃЕсЃАсЃ╝сѓ┐сѓёсѓфсЃЌсѓисЃДсЃ│сѓњуц║сЂЎсЂЪсѓЂсЂФСй┐ућесЂЋсѓїсЂЙсЂЎсђѓ
    • СЙІ№╝џcommand [option] сЂ»сђЂсѓфсЃЌсѓисЃДсЃ│сѓњС╗╗ТёЈсЂДТЈљСЙЏсЂДсЂЇсѓІсЂЊсЂесѓњуц║сЂЌсЂдсЂёсЂЙсЂЎсђѓСЙІсЂѕсЂ░ command -v сѓё command сЂ«сѓѕсЂєсЂФсЂЌсЂЙсЂЎсђѓ
  3. Т│бТІгт╝Д {}№╝џ
    • Т│бТІгт╝ДсЂ»жђџтИИсђЂжЂИТіъУѓбсѓётђцсЂ«сѓ░сЃФсЃ╝сЃЌсѓњУАесЂЎсЂ«сЂФСй┐ућесЂЋсѓїсЂЙсЂЎсђѓсЂцсЂЙсѓісђЂСИјсЂѕсѓЅсѓїсЂЪжЂИТіъУѓбсЂ«СИГсЂІсѓЅ1сЂцсѓњжЂИсЂ░сЂфсЂЉсѓїсЂ░сЂфсѓЅсЂфсЂёсЂЊсЂесѓњТёЈтЉ│сЂЌсЂЙсЂЎсђѓсЂЊсѓїсЂ»сђїжЂИТіъуџёсЃЉсЃЕсЃАсЃ╝сѓ┐сѓ░сЃФсЃ╝сЃЌсђЇсЂесѓѓтЉ╝сЂ░сѓїсЂЙсЂЎсђѓ
    • СЙІ№╝џcommand {option1 | option2 | option3} сЂ»сђЂСИјсЂѕсѓЅсѓїсЂЪжЂИТіъУѓбсЂ«СИГсЂІсѓЅ1сЂцсѓњжЂИсЂ░сЂфсЂЉсѓїсЂ░сЂфсѓЅсЂфсЂёсЂЊсЂесѓњуц║сЂЌсЂдсЂёсЂЙсЂЎсђѓСЙІсЂѕсЂ░ command option2 сЂ«сѓѕсЂєсЂФсЂЌсЂЙсЂЎсђѓ
  4. ТІгт╝Д ()№╝џ
    • сѓ│сЃъсЃ│сЃЅсЃЕсѓцсЃ│сЃўсЃФсЃЌсЂДсЂ»сђЂТІгт╝ДсЂ»жђџтИИсЃЉсЃЕсЃАсЃ╝сѓ┐сЂ«сѓ░сЃФсЃ╝сЃЌтїќсѓњУАесЂЎсЂ«сЂФСй┐ућесЂЋсѓїсђЂсЃЉсЃЕсЃАсЃ╝сѓ┐сЂ«ТДІжђасЂетёфтЁѕжаєСйЇсѓњТўјуб║сЂФсЂЌсЂЙсЂЎсђѓ
    • СЙІ№╝џcommand (option1 | option2) filename сЂ»сђЂoption1 сЂЙсЂЪсЂ» option2 сЂ«сЂёсЂџсѓїсЂІсѓњжЂИТіъсЂЌсђЂсЃЋсѓАсѓцсЃФтљЇсѓњсЃЉсЃЕсЃАсЃ╝сѓ┐сЂесЂЌсЂдТЈљСЙЏсЂЎсѓІт┐ЁУдЂсЂїсЂѓсѓІсЂЊсЂесѓњуц║сЂЌсЂдсЂёсЂЙсЂЎсђѓСЙІсЂѕсЂ░ command option1 file.txt сЂ«сѓѕсЂєсЂФсЂЌсЂЙсЂЎсђѓ
