гғҗгғјгӮёгғ§гғіиҰҒд»¶

зҸҫеңЁгҒ®гғҗгғјгӮёгғ§гғігӮ№гғҶгғјгӮҝгӮ№пјҡ

  • жңҖж–°е®үе®ҡзүҲпјҡ2.5.9пјҲгғҚгғғгғҲгғҜгғјгӮҜиЁӯе®ҡж¬ йҷҘгҒӮгӮҠпјү
  • жҺЁеҘЁгғҗгғјгӮёгғ§гғіпјҡ2.6.0 гғ—гғ¬гғ“гғҘгғјзүҲпјҲе®Ңе…ЁmirroredгғўгғјгғүгӮөгғқгғјгғҲпјү

гғўгғјгғүжҜ”ијғеҲҶжһҗ

зү№еҫҙbridgeгғўгғјгғүпјҲе»ғжӯўдәҲе®ҡпјүmirroredгғўгғјгғүпјҲжҺЁеҘЁпјү
гғ—гғӯгғҲгӮігғ«гӮ№гӮҝгғғгӮҜгӮўгғјгӮӯгғҶгӮҜгғҒгғЈгғҮгғҘгӮўгғ«гғ—гғӯгғҲгӮігғ«гӮ№гӮҝгғғгӮҜгӮ·гӮ§гӮўгғ—гғӯгғҲгӮігғ«гӮ№гӮҝгғғгӮҜ
IPгӮўгғүгғ¬гӮ№еүІгӮҠеҪ“гҒҰзӢ¬з«ӢIPпјҲWindows+WSLпјүгғӣгӮ№гғҲIPе…ұжңү
гғқгғјгғҲгғӘгӮҪгғјгӮ№зӢ¬з«ӢдҪҝз”ЁгғқгғјгғҲе…ұжңүпјҲз«¶еҗҲеӣһйҒҝеҝ…иҰҒпјү
гғҚгғғгғҲгғҜгғјгӮҜжҖ§иғҪжҜ”ијғзҡ„йҮҚгҒ„и»ҪйҮҸеҠ№зҺҮзҡ„
иЁӯе®ҡиӨҮйӣ‘еәҰз°ЎеҚҳгғ•гӮЎгӮӨгӮўгӮҰгӮ©гғјгғ«гғқгғӘгӮ·гғјгҒ®ж·ұгҒ„иЁӯе®ҡеҝ…иҰҒ

жЁҷжә–иЁӯе®ҡжүӢй Ҷ

1. гғҚгғғгғҲгғҜгғјгӮҜгғўгғјгғүиЁӯе®ҡ

WSL Settings гӮўгғ—гғӘгҒ§еҹәзӨҺгғўгғјгғүгӮ’иЁӯе®ҡпјҡ

  1. иЁӯе®ҡгӮўгғ—гғӘгӮ’й–ӢгҒҸ
  2. “Network"гӮҝгғ–гӮ’йҒёжҠһ
  3. гғҚгғғгғҲгғҜгғјгӮҜгғўгғјгғүгӮ’"Mirrored"гҒ«иЁӯе®ҡ
  4. иЁӯе®ҡгӮ’йҒ©з”ЁгҒ—WSLгӮ’еҶҚиө·еӢ•

2. гғ•гӮЎгӮӨгӮўгӮҰгӮ©гғјгғ«гғқгғӘгӮ·гғјиЁӯе®ҡ

PowerShellгҒ§е®Ңе…ЁгғқгғӘгӮ·гғјиЁӯе®ҡгӮ’е®ҹиЎҢпјҡ

# WSLд»®жғігғһгӮ·гғіGUIDе®ҡзҫ©
$wslGuid = '{40E0AC32-46A5-438A-A0B2-2B479E8F2E90}'

# гғ•гӮЎгӮӨгӮўгӮҰгӮ©гғјгғ«гғқгғӘгӮ·гғјиЁӯе®ҡпјҲй ҶеәҸйҖҡгӮҠе®ҹиЎҢпјү
Set-NetFirewallHyperVVMSetting -Name $wslGuid -Enabled True
Set-NetFirewallHyperVVMSetting -Name $wslGuid -DefaultInboundAction Allow
Set-NetFirewallHyperVVMSetting -Name $wslGuid -DefaultOutboundAction Allow
Set-NetFirewallHyperVVMSetting -Name $wslGuid -LoopbackEnabled True
Set-NetFirewallHyperVVMSetting -Name $wslGuid -AllowHostPolicyMerge True

# иЁӯе®ҡзөҗжһңжӨңиЁј
Get-NetFirewallHyperVVMSetting -Name $wslGuid

3. гғқгғјгғҲгғһгғғгғ”гғігӮ°жӨңиЁј

# дҫӢпјҡ80з•ӘгғқгғјгғҲдҪҝз”ЁзҠ¶жіҒгӮ’зўәиӘҚ
Get-NetTCPConnection -LocalPort 80

гӮҲгҒҸгҒӮгӮӢе•ҸйЎҢеҜҫеҮҰ

е•ҸйЎҢ1пјҡеӨ–йғЁжҺҘз¶ҡгҒҢзўәз«ӢгҒ§гҒҚгҒӘгҒ„

  • зўәиӘҚжүӢй ҶпјҡGet-NetFirewallHyperVVMSettingеҮәеҠӣгҒ§е…ЁгҒҰгҒ®гғ•гӮЈгғјгғ«гғүгҒҢTrue/AllowгҒ§гҒӮгӮӢгҒ“гҒЁ
  • и§Јжұәзӯ–пјҡгғ•гӮЎгӮӨгӮўгӮҰгӮ©гғјгғ«гғқгғӘгӮ·гғјиЁӯе®ҡгӮ’й ҶеәҸйҖҡгӮҠеҶҚе®ҹиЎҢ

е•ҸйЎҢ2пјҡгғқгғјгғҲз«¶еҗҲ

  • жӨңиЁјж–№жі•пјҡnetstat -anoгҒ§гғқгғјгғҲдҪҝз”ЁзҠ¶жіҒгӮ’зўәиӘҚ
  • еҜҫеҮҰжҸҗжЎҲпјҡWindowsеҒҙгғқгғјгғҲгӮ’е„Әе…Ҳзҡ„гҒ«и§Јж”ҫгҖҒгҒҫгҒҹгҒҜWSLгӮөгғјгғ“гӮ№гҒ®гғӘгӮ№гғігғқгғјгғҲгӮ’еӨүжӣҙ

жӨңиЁјж–№жі•

  1. WSLгӮөгғјгғ“гӮ№пјҲNginx/ApacheзӯүпјүгӮ’иө·еӢ•
  2. WindowsгғӣгӮ№гғҲгҒӢгӮүhttp://localhost:<port>гҒ«гӮўгӮҜгӮ»гӮ№
  3. гғӯгғјгӮ«гғ«гғҚгғғгғҲгғҜгғјгӮҜгғҮгғҗгӮӨгӮ№гҒӢгӮүhttp://<host-ip>:<port>гҒ«гӮўгӮҜгӮ»гӮ№

еҸӮиҖғиіҮж–ҷ

  1. WSLе…¬ејҸгғҚгғғгғҲгғҜгғјгӮҜгғүгӮӯгғҘгғЎгғігғҲ
