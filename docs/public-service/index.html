Public Service

Tags:
  • Tutorial
  • Public Service
Categories:
  • Tutorial

First, it must be noted that some home networks prohibit commercial use under their service agreements. In practice, the definition of commercial use is not enforced by whether it is actually commercial. Once inbound HTTP/HTTPS connections exceed a certain number (some say 250), the service is downgraded, which significantly affects the network. Therefore, any service intended for public access must be treated as a commercial service. It is best to rent a VPS from a cloud provider to offer public services.
