par ex.

NotationDescription
Texte sans crochets ou accoladesÉléments que vous devez saisir tels qu’affichés.
<Texte entre chevrons>Espace réservé pour lequel vous devez fournir une valeur.
[Texte entre crochets carrés]Éléments facultatifs.
{Texte entre accolades}Ensemble d’éléments requis. Vous devez en choisir un.
Barre verticale ( | )Séparateur d’éléments mutuellement exclusifs. Vous devez en choisir un.
Ellipse (…)Éléments pouvant être répétés et utilisés plusieurs fois.
